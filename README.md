# Music Collection

This is a backend application for managing the music-library like artists and albums.

## Tech-Stack

### Development
- Kotlin
- Quarkus
- Hibernate
- Flyway
- PostgreSQL
- Keycloak
- MinIO

### Test
- RestAssured
- Mockito
- Kotest
- Docker-Testcontainer

### Buildsystem
- Gradle

### CI/CD
- GitHub Actions

## Install with Docker
Create and start the containers:
```bash
docker compose up -d
```
